# WoW Addon Patches

Custom patch and compatibility module for World of Warcraft (Anniversary Edition) addon [HealPredict TBC Anniversary + Classic](https://www.curseforge.com/wow/addons/healpredict-tbc-anniversary)

## HealPredict - SUF Compatibility

Adds full Shadowed Unit Frames support to HealPredict. See [SUF_README.md](HealPredict/SUF_README.md) for details and installation instructions.

**Modified files:**
- `HealPredict/Main/Modules/SUFCompat.lua`
- `HealPredict/Main/Modules/Modules.xml`
- `HealPredict/Main/Config/Init.lua`
